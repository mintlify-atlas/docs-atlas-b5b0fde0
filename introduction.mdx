listmonk is a self-hosted, high performance one-way mailing list and newsletter manager. It comes as a standalone binary and the only dependency is a PostgreSQL database.

## What is listmonk?

listmonk is a standalone, self-hosted, newsletter and mailing list manager. It is fast, feature-rich, and packed into a single binary. It uses a PostgreSQL database as its data store.

## Key Features

- **Self-hosted**: Complete control over your data and infrastructure
- **High performance**: Optimized for handling large mailing lists with millions of subscribers
- **Single binary**: Easy deployment with no complex dependencies
- **PostgreSQL database**: Reliable and scalable data storage
- **Rich templating**: Powerful Go template expressions for dynamic HTML emails
- **Multiple messengers**: Support for SMS, FCM notifications, and more beyond email
- **Privacy-focused**: GDPR-compliant tracking options with anonymous analytics
- **API-driven**: Full REST API for integrations and automation

## Architecture

listmonk follows a simple architecture:

- **Backend**: Written in Go for performance and reliability
- **Frontend**: Built with Vue.js and Buefy UI components
- **Database**: PostgreSQL 12 or higher
- **Deployment**: Available as a binary or Docker container
