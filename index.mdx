    <p className="text-base dark:text-gray-400 text-gray-600">Get Listmonk running in minutes</p>
  </div>
  
  <Steps>
    <Step title="Install with Docker">
      Download and run Listmonk using Docker Compose:
      
      ```bash
      # Download the compose file
      curl -LO https://github.com/knadh/listmonk/raw/master/docker-compose.yml
      
      # Run the services
      docker compose up -d
      ```
      
      Visit `http://localhost:9000` to access your Listmonk instance.
    </Step>
    
    <Step title="Create your first list">
      Navigate to **Lists** in the dashboard and create a new mailing list. Choose between single opt-in (subscribers are immediately added) or double opt-in (subscribers must confirm via email).
    </Step>
    
    <Step title="Import subscribers">
      Go to **Subscribers** → **Import** to bulk import your subscriber list via CSV or API. Map your CSV columns to subscriber attributes and assign subscribers to lists.
    </Step>
    
    <Step title="Send your first campaign">
      Create a campaign under **Campaigns**, select your target lists, design your email with the rich text editor or HTML, and send or schedule it.
    </Step>
  </Steps>
</div>
